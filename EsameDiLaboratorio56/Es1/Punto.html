Punto Submit Solution

Creare i file punto.h e punto.c che consentano di utilizzare le seguenti strutture:

  1. struct punto_cart {
  2.     double x, y;
  3. };
  4. struct punto_pol {
  5.     double r, t;
  6. };

e le seguenti funzioni:

  1. extern struct punto_pol cartesiano2polare(const struct punto_cart *p);
  2. extern struct punto_cart polare2cartesiano(const struct punto_pol *p);

La prima funzione accetta un puntatore a un punto in coordinate cartesiane e deve restituire una struct punto_pol contenente il punto p convertito in coordinate polari. Le formule per la conversione sono le seguenti:

r=x2+y2t=arctan⁡2(y,x)r=\sqrt{x^2+y^2}\\ t=\arctan2(y,x)r=x2+y2​t=arctan2(y,x)

La seconda funzione accetta un puntatore a un punto in coordinate polari e deve restituire una struct punto_cart contenente il punto p convertito in coordinate cartesiane. Le formule per la conversione sono le seguenti:

x=rcos⁡(t)y=rsin⁡(t)x=r\cos(t)\\ y=r\sin(t)x=rcos(t)y=rsin(t)

Gli angoli sono sempre espressi in radianti. I punti passati alle funzioni non saranno mai NULL. La funzione arctan⁡2()\arctan2()arctan2() è disponibile nella libreria <math.h> con il nome di atan2(). Attenzione all'ordine di x e y da passare a atan2()!
